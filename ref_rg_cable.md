# RG Cable Guide

* RG6
	* 75 ohms
	* OD: 0.27"
	* cable television, satellite television, cable modems
* RG58
	* 50 ohms
	* OD: 0.195"
	* radio communication
* RG213
	* 50 ohms
	* OD: 0.405"
	* radio communication
* LMR-400
	* 50 ohms
	* OD: ???
	* radio communication
